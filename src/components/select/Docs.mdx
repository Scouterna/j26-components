# Select

The Select component is used to select a value from a list of options. It
extends the [Radix Select component](https://www.radix-ui.com/primitives/docs/components/select).

A basic Select component is comprised of the following parts:
- `Select.Root`: An invisible wrapper in which all must live.
- `Select.Trigger`: The button that opens the select menu.
- `Select.Value`: The value that is displayed in the trigger.
- `Select.Content`: The menu that opens when the trigger is clicked.
- `Select.Item`: The individual items in the menu.

Optionally you can also group items in the menu using:
- `Select.SelectGroup`: A group of items in the menu.
- `Select.Label`: A label for a group of items in the menu.

Here is a basic example of a Select component:
```tsx
import { Select } from '@scouterna/ui';

<Select.Root>
  <Select.Trigger>
    <Select.Value placeholder="Select an option" />
  </Select.Trigger>
  <Select.Content>
    <Select.Item value="option1">Option 1</Select.Item>
    <Select.Item value="option2">Option 2</Select.Item>
    <Select.Item value="option3">Option 3</Select.Item>
  </Select.Content>
</Select.Root>
```
